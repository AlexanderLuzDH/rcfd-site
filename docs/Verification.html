Verification of downloads (interim)

Until signed Windows wheels are published, you can verify artifacts manually:

Wheels (hash)

  • Download the wheel and compute a SHA256 hash locally.
  • Compare against a published hash in the release notes or CI artifact logs.

Example (PowerShell)

powershell Get-FileHash rcfd_pcg-0.0.2-py3-none-any.whl -Algorithm SHA256 | Format-List

Example (POSIX)

bash shasum -a 256 rcfd_pcg-0.0.2-py3-none-any.whl

Docker image (digest)

  • When the CPU image is published, pull by digest: bash docker pull yourorg/rcfd-pcg@sha256:<digest>
  • Check docker image inspect and confirm the digest matches the release notes.
