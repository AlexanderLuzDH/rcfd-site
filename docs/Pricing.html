Pro (No Support)

  • Token format: signed JSON payload (offline) with expiry + feature bits

  • Signature: HMAC‑SHA256 over canonical JSON; verification is fully offline. No telemetry, no pings.
  • Unlocks that add value without support burden:
  • Block‑diagonal preconditioner presets and guardrails (bigger‑d friendly)
  • Large‑run helpers (suite export; consolidated proof‑pack across many configs)
  • Extended proof‑pack (plots embedded, comparison tables, artifact signing)
  • Spill‑to‑disk controls and concurrency knobs (resource efficiency)
