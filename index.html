Quickstart

Product promise: Free, self-serve. No support, no meetings. Upgrade to Pro to unlock advanced features — still no support. If the proof-pack doesn’t show wins in our recommended regimes, don’t buy.

pip install rcfd-pcg==0.1.0
python -m rcfd_pcg.cli demo --out results/demo_report.json --html results/demo_report.html
python scripts/proof_pack_pipeline.py
# Then open: results/proof_pack.html
